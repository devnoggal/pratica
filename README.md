# PRATICA
Repositório pessoal para treinar lógica de programação através da resolução de exercícios e pequenos desafios de código.

# Objetivo
Este repositório serve como espaço de estudo e prática contínua, reunindo exercícios resolvidos em Java com foco em resolução de problemas lógicos em java de forma geral.


# Organização das branches
main — branch estável do repositório, contém apenas a documentação do projeto.
develop — branch de desenvolvimento, onde os exercícios são adicionados e evoluídos.


# Estrutura
Cada exercício é organizado em um arquivo .java separado, nomeado de forma sequencial (ex1.java, ex2.java, ...), com o problema resolvido diretamente no código.

├── ex1.java
├── ex2.java
├── ex3.java
├── ex4.java
├── ex5.java
├── README.md
└── LICENSE

# Como executar
Cada exercício pode ser compilado e executado individualmente com o JDK instalado:

bashjavac ex1.java
java ex1

# Sobre
Este é um projeto pessoal de estudo, sem pretensão de produção. O foco é a prática constante e a evolução gradual na resolução de problemas lógicos.
